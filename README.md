# Threaded Artifact Protocol - TAP

Threaded Artifact Protocol is a sophisticated context engineering Android application, that approaches the context engineering for LLMs in a social media like interface, instead of traditional chat interface.

### About the App

This app is built on MVI - Model View Intent Architecure , that connects to local LLMs over the **network** 

Utilizes social media like "comments" UI to do folllow up interaction.
